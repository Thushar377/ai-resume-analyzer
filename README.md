# AI Resume Analyzer & Job Matcher

AI-powered web application where users can upload resumes and receive skill extraction, AI feedback, and job-match analysis.

## Features

- User Registration & Login
- JWT Authentication
- Resume Upload: PDF/DOCX
- PostgreSQL Database
- AI Skill Extraction
- Job Description Input
- AI Match Score Generation
- Dashboard with Previous Analyses
- REST API with Swagger Documentation
- FastAPI + Jinja2 Frontend

## Tech Stack

- Backend: FastAPI, Python, FastAPI Templates
- Frontend: HTML, Tailwind CSS, JavaScript, Jinja2
- Database: PostgreSQL
- Authentication: JWT
- AI: OpenAI API
- Documentation: Swagger UI
